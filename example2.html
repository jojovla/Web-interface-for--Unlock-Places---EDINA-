Example 2: Search a feature type within a location

1. If you want to find a feature type within a region, click on the orange button, on the right side of the page. A panel with options will appear. Write the feature type you want on the text box "Feature type" and select the option of the autocomplete result. Then there are two choices available: either type a location in the equivalent text box or press the "Draw rectangle" button to define on the map a specific region. In both cases, the results appears as vectors inside a bounding box.

2. Besides location, a place can be searched by inserting latitude and longitude coordinates on the coordinates text box or by typing a postcode on the equivalent text box. In both cases, when pressing the Enter key in a textbox, the browser will automatically display the resutls.
